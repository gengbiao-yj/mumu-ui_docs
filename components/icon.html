iconify 图标组件 #

介绍 #

基于 iconify 的图标组件，支持图标库：iconify

代码演示 #

基本使用 #

通过 icon 属性来指定需要使用的图标，默认值为 'ep:add-location'
<template>
  <mumu-icon height="2em" icon="ep:add-location" width="2em" />
</template>

<template>
  <mumu-icon height="2em" icon="ep:add-location" width="2em" />
</template>

图标颜色 #

通过 color 属性来设置图标的颜色。
<template>
  <div class="mumu-icon-list">
    <mumu-icon
      v-for="(icon, i) in icons"
      :key="i"
      :color="icon.color"
      :height="icon.height"
      :icon="icon.icon"
      :width="icon.width"
    />
  </div>
</template>

      :key="i"
      :color="icon.color"

      :icon="icon.icon"
      :width="icon.width"

图标模式 #

phone inline:true;phone inline:false
通过 inline 属性来设置图标的内联或块模式。区别于块模式，内联模式的图标通过设置 css vertical-align 样式与文本对齐。默认为块模式
<template>
  <div>
    <mumu-icon
      color="#00a8ff"
      height="32px"
      icon="ep:add-location"
      :inline="true"
      width="32px"
    />
    <span>phone inline:true;</span>
    <mumu-icon
      color="#fbc531"
      height="32px"
      icon="ep:add-location"
      :inline="false"
      width="32px"
    />
    <span>phone inline:false</span>
  </div>
</template>

<script lang="ts" setup></script>

  div {
    svg {
      display: inline-block;
      margin: 1px;
      border-bottom: 4px solid #eca5a5;
    }
  }
</style>

图标旋转 #

通过 rotate 属性旋转图标。可选角度为 90、180、270。
<template>
  <div>
    <label>数值是1代表90度，2代表180度，3代表270度。</label>
    <mumu-icon
      color="#00a8ff"
      height="32px"
      icon="ep:bicycle"
      :inline="true"
      width="32px"
    />
    <mumu-icon
      color="#fbc531"
      height="32px"
      icon="ep:bicycle"
      :inline="true"
      :rotate="1"
      width="32px"
    />

      color="#fbc531"
      height="32px"
